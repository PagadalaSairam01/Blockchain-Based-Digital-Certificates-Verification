# Blockchain-Based-Digital-Certificates-Verification
This project implements a secure and decentralized certificate validation system using blockchain technology. It addresses the growing problem of certificate forgery by leveraging Ethereum-inspired blockchain architecture, cryptographic hashing (SHA-256), and a Python-based GUI to securely issue, store, and verify digital certificates.<br>


<h2>Features</h2>
Blockchain-secured certificate storage<br>
Tamper-proof validation<br>
Smart contract-based transparent verification<br>
GUI for certificate upload and verification<br>
Python-based implementation<br>
Fast verification with no central authority dependency<br><br>

<h2>How It Works</h2><br>
<H2>Certificate Issuance</H2><br>
User inputs Roll No, Name, Contact<br>
Uploads certificate (PDF)<br>
SHA-256 digital signature is generated<br>
Block added to blockchain<br>
Ledger updated & saved locally<br>
<H2></H2>Certificate Verification</H2><br>
User uploads a certificate for verification<br>
System computes hash<br>
Hash is compared with blockchain records<br>
Displays "Verification Successful" or "Verification Failed"<br>

<h2>Technology Stack</h2><br>
Programming Language: Python 3.7<br>
Frameworks/Libraries: Tkinter, hashlib, pickle<br>
Blockchain Layer: Custom Python blockchain inspired by Ethereum architecture<br>
Hashing Algorithm: SHA-256<br>
